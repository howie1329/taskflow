# New Product Ideas - Quick Reference

## 🏆 Top 5 Most Fun to Build

### 1. **AI Code Review Assistant** ⭐⭐⭐⭐⭐
**What**: AI analyzes PRs, suggests improvements, learns your codebase patterns
**Why Fun**: AST parsing + RAG + real-time feedback = technically interesting
**MVP Time**: 2-3 weeks
**Stack**: Your existing stack + code analysis libraries

### 2. **AI Research Assistant & Knowledge Base** ⭐⭐⭐⭐⭐
**What**: Research tool with vector search, knowledge graphs, citation tracking
**Why Fun**: Perfect RAG use case, visual knowledge graphs, you'd use it
**MVP Time**: 3-4 weeks
**Stack**: Web scraping + PDF parsing + your RAG expertise

### 3. **AI Meeting Notes Extractor** ⭐⭐⭐⭐
**What**: Real-time transcription → action items → task creation
**Why Fun**: Real-time audio + AI extraction + integrations
**MVP Time**: 2-3 weeks
**Stack**: Whisper API + your AI tools + calendar APIs

### 4. **AI Writing Coach** ⭐⭐⭐⭐
**What**: Real-time writing assistant that learns your style
**Why Fun**: Streaming suggestions + style learning + NLP challenges
**MVP Time**: 2-3 weeks
**Stack**: Your AI stack + writing analysis

### 5. **AI Recipe Generator** ⭐⭐⭐⭐
**What**: "What can I make with these ingredients?" → recipes + meal plans
**Why Fun**: Practical, multi-modal (images), RAG for preferences
**MVP Time**: 2-3 weeks
**Stack**: Image recognition + recipe generation + your stack
